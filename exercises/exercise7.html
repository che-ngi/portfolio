HackHarvard 2019
In 2019, I attended a hackathon(Design sprint-like event, "Hack Marathon") hosted by Harvard University. My team and I created a Web/Mobile App called アニメCam (Anime Cam) that gives you information regarding an anime(Japanese Animation) by taking a photo or screenshot. It would give information such as the name of the anime, the episode, the time stamp, as well as other fun facts about the anime. We used a Python Flask server in the back end which has an endpoint which takes the parameter of an image link. There is a web page which was built with HTML and CSS from scratch which touches our endpoint. The application itself was made using React Native and Expo. The anime search functionality uses the API Trace.moe. The image uploading is done through converting the binary to base64 and then pushing to Imgur using the Imgur API.
